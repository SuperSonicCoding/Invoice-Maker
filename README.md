# Invoice-Maker
Invoice maker for businesses, specifically my parent's business.

# Requirements:
* Be able to change:
  * Invoice number
  * Date (preset day delivered)
  * Quanitity
  * Description
  * TO field (company being sent to)
  * Address
  * Unit Price
* Lego on invoice
* To: field
* From: field
* All invoices of company within a folder
* Editable folder location for company
* Each company has it's own profile
* Have a database that holds each company
